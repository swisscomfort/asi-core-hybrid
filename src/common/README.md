# ASI Common Module

Dieses Modul enthält grundlegende Utilities, Logging und Hilfsfunktionen, die von allen anderen Modulen verwendet werden können.

## Zweck

- Gemeinsame Utilities und Helper-Funktionen
- Logging-Infrastruktur
- Grundlegende Datenstrukturen
- Error-Handling Mechanismen

## API-Übersicht

- `asi::common::Logger` - Zentrales Logging
- `asi::common::Config` - Konfigurationsmanagement
- `asi::common::Utils` - Allgemeine Hilfsfunktionen
